Zataone PolicyLens

Visual compliance review for demos — Quick (VLM → LLM vs policy) or Full (extractors + LLM vs policy; optional YAML rule engine). Same APIs: upload → poll → graph.

Sent on every API request. The service must allow this name (Cloud Run env ZATAONE_ALLOWED_DOMAINS, comma-separated; the default domain is always allowed). If you pick a domain the server does not allow, the API returns 403.

Pipeline:

Quick: no extractors, rule engine off — outcome from Gemini VLM + text LLM vs policy clauses. Full: OCR/vision signals + LLM vs policy; set API env ZATAONE_POLICY_ENGINE_ENABLED=true only if you want YAML rule matching too.
